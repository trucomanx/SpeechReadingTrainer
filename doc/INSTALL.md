# speech-reading-trainer

A desktop application that helps users improve reading fluency and pronunciation by combining text-to-speech playback, voice recording, automatic transcription, and real-time accuracy feedback.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/speech_reading_trainer/

```bash
pip install --upgrade speech_reading_trainer
```

Using:

```bash
speech-reading-trainer
```

## Install from source
Installing `speech-reading-trainer` program

```bash
git clone https://github.com/trucomanx/SpeechReadingTrainer.git
cd SpeechReadingTrainer
pip install -r requirements.txt
cd src
python -m build
pip install dist/speech_reading_trainer-*.tar.gz
```
Using:

```bash
speech-reading-trainer
```

## Uninstall

```bash
pip uninstall speech_reading_trainer
```
